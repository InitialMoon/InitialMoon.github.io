# InitialMoon.github.io

Personal homepage for Yingqi Jie (介应奇), hosted at https://initialmoon.github.io/.

The site is built with HugoBlox Academic CV and presents research, publications, selected projects, and CV information for a research + engineering personal homepage.

## Development

Install dependencies:

```bash
pnpm install
```

Preview locally:

```bash
hugo server --disableFastRender
```

Build the static site:

```bash
hugo --minify
```

Generate the local Pagefind search index after building:

```bash
pnpm run pagefind
```
